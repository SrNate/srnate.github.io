📖 Cómo modificar este calendario

  • Busca el bloque const HORARIO = { ... } en el script para editar cada día.
  • Cada entrada tiene: type, gym, note, exams y blocks.
  • Para añadir un examen: añade un objeto en el array exams del día correspondiente.
  • Para cambiar un bloque de estudio: edita el array blocks del día (time, subject, label, detail).
  • Los subjects válidos son: lengua, mates, fisica, dibujo, historia, ingles, filosofia, gym, libre, clase.
  • Para cambiar colores de asignaturas: edita las variables CSS en :root { --c-lengua: ... }.
  • Los exámenes de Filosofía marcados con proposed: true aparecen con borde discontinuo — quíta ese flag cuando se confirme la fecha.
